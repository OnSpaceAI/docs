### What is Spaces?

**Spaces** is OnSpace's built-in backend service that provides a complete serverless backend solution for your React Native applications.

Think of it as **having a complete backend infrastructure that materializes instantly** when you need it, without any of the traditional setup headaches.

### Core Features

**🔐 User Authentication**

- OTP and password-based login
- Session management
- Social auth providers

**🗄️ PostgreSQL Database**

- Fully managed database instance
- Row Level Security (RLS) built-in
- Real-time data sync

**📁 File Storage**

- Images, documents, and media handling
- CDN-optimized delivery
- Automatic optimization

**⚡ Edge Functions**

- Serverless API endpoints
- Custom business logic
- Auto-scaling execution

**📊 Visual Dashboard**

- Data management interface
- User administration
- Real-time monitoring

### Why Choose Spaces?

**Instant Backend Setup**

- Zero server configuration
- Database automatically provisioned
- Authentication working out-of-the-box

**Developer-Friendly**

- Same APIs and SDKs as Supabase
- Full TypeScript support
- Seamlessly integrated with OnSpace AI

**Visual Management**

- Built-in dashboard for data operations
- User management interface
- Real-time performance monitoring
