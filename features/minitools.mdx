### **What is mini tools?**

Mini Tools is a collection of useful community-contributed mini applications designed to help builders streamline their development workflow. These lightweight tools solve common challenges developers face when building websites and apps.

## **UploadNow**

**Solve CDN link issues:** Upload images, audio, and video files and get fast-loading CDN links powered by Cloudflare for your projects.

**ðŸ”— Try it now:** https://uploadnow.onspace.build/

### **What It Does**

UploadNow helps you handle media files efficiently by providing fast, reliable CDN links for your website assets. Instead of dealing with slow-loading files or complex hosting setups, simply upload your files and get optimized CDN URLs.

### **Common Use Cases**

- **Website logos:** Upload your company logo and get a fast-loading link
- **Game development:** Add background music (BGM) to your games with reliable audio links
- **Landing pages:** Include explanatory videos that load quickly for better user experience
- **Media galleries:** Host images, videos, and audio with global fast delivery
- **Static assets:** Any media file that needs to be displayed on your website

### **How It Works**

1. **Upload Your Files**
   - Drag and drop or browse to select your files
   - Supports: Images (JPG, PNG, GIF, SVG), Audio (MP3, WAV), Video (MP4, MOV)
2. **Automatic Processing**
   - Files are automatically uploaded to Cloudflare's global CDN network
   - Optimized for fast loading worldwide
3. **Get CDN Links**
   - Receive secure, fast-loading CDN URLs for each uploaded file
   - Copy the links for use in your projects
4. **Use in OnSpace**
   - Simply paste the CDN link into OnSpace chat
   - Tell the agent where to place the file (e.g., "Use this link for the hero section background image")

### **Benefits**

- **Fast Loading:** Global CDN ensures quick file delivery
- **Reliable:** 99.9% uptime with Cloudflare infrastructure
- **Easy Integration:** No complex setup - just upload and use
- **Cost Effective:** Free tier available for most use cases
- **Global Reach:** Files load quickly worldwide

### **Getting Started**

1. Visit https://uploadnow.onspace.app
2. Upload your media files
3. Copy the generated CDN links
4. Use the links in your OnSpace projects
