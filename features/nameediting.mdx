Step-by-Step Instructions
Navigate to Your Project

Open the project you want to rename
Access the Project Name

Look for the project name displayed in the top-left corner of the project interface
Click on the project name
Select Edit Option

A dropdown menu will appear
Click on "Edit Name" from the menu options
Rename Your Project

A popup window will open with the current project name
Clear the existing name or modify it as needed
Enter your desired new project name
Save Changes

Click "Save" or confirm the changes
The popup window will close automatically
Your project name will be updated immediatel
